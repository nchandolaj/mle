# The Pythonic Standard

*Focus: Writing clean, idiomatic code and understanding complexity.*

## Whats Covered?
* [**The Big O Notation**](https://github.com/nchandolaj/mle/blob/main/cs%20&%20data/docs/big_o_notation.md): Discuss Time vs. Space complexity. Understand why $O(1) < O(\log n) < O(n) < O(n \log n) < O(n^2) < O(n!)$.
* [**The Space-Time Trade-off**](https://github.com/nchandolaj/mle/blob/main/cs%20%26%20data/docs/space_time_trade_off.md): Discuss the Space-Time Trade-offs. Review some common real-world examples.
* [**Python Internals:**](https://github.com/nchandolaj/mle/blob/cs-%26-data/cs%20%26%20data/docs/python_internals.md): 
        * List vs. Tuple vs. Set (Lookup times).
        * `dict` implementation (Hash map).
        * List slicing `arr[start:stop:step]` (Note: Slicing creates a copy, costing $O(k)$ memory).
* [**Example: Two-Sum Problem**](https://github.com/nchandolaj/mle/blob/main/cs%20%26%20data/docs/two_sum_problem.md): Discuss different approaches explaining their space-time trade-offs.
